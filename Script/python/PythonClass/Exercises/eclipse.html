Using Python with Eclipse

Objectives:
  • Learn how edit and run programs in Eclipse, using pydev

Presentation slides for this section can be in PythonClass/Optional/Eclipse.pdf.

pydev is a plugin that allows Python programs to be edited in the popular Eclipse IDE. In this exercise, simply see if you can re-do the first two exercises using the Eclipse environment. To do this, you will need to figure out how to start the interactive Python interpreter and how to edit, save, and run programs.
