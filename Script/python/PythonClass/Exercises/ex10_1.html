(a) A simple example of variable arguments

Try defining the following function:

>>> def avg(x,*more):
        return float(x+sum(more))/(1+len(more))

>>> avg(10,11)
10.5
>>> avg(3,4,5)
4.0
>>> avg(1,2,3,4,5,6)
3.5
>>>

Notice how the parameter *more collects all of the extra arguments.

(b) Passing a tuple as arguments

Suppose you read some data from a file and obtained a tuple such as this: 

>>> data = ('GOOG',100,490.10)
>>>

Now, suppose you wanted to create a Stock object from this data. Try the following statement to see an easy way to do it: 

>>> from stock import Stock
>>> s = Stock(*data)
>>>

(c) Creating a bunch of instances

Try reading a portfolio into a list of dictionaries using either your report.py program or fieldparse.py. For example: 

>>> import report
>>> portfolio = report.read_portfolio("Data/portfolio.dat")
>>> for s in portfolio:
       print s

... inspect the contents ...
>>>

Suppose you wanted to convert the entire list into a list of Stock instances. Here's one way to do it: 

>>> portfolio = [Stock(**s) for s in portfolio]
>>> for s in portfolio:
       print s

... look at the output ...
>>>

In this example, the **s turns dictionaries into keyword arguments that are passed to the Stock.__init__() function.
