(a) Tuples

At the interactive prompt, create the following tuple that represents an investment of stock: 

>>> s = ('GOOG', 100, 490.10)
>>> print s
('GOOG', 100, 490.1)
>>>
If the 2nd field is the number of shares and the 3rd field is the price per share, calculate the total cost of the holding by multiplying these two fields together: 
>>> cost = s[1]*s[2]
>>> cost
49010.0
>>>
Tuples are read-only. Verify this by trying to change the number of shares to 75. 
>>> s[1] = 75
(see what happens. hint: it doesn't work)
>>>
Although you can't change tuple contents, you can always create a completely new tuple that replaces the old one. Try this: 
>>> s = (s[0],75,s[2])
>>> s
('GOOG', 75, 490.1)
>>>

Note: When you make a new tuple and reassign the variable name, the old tuple is discarded and garbage collected.

Tuples are often used to pack and unpack values into variables. Try the following:

>>> name, shares, price = s
>>> name
'GOOG'
>>> shares
75
>>> price
490.1
>>>
Take the above variables and pack them back into a tuple 
>>> s = (name, 2*shares, price)
>>> s

(b) Dictionaries as a data structure

At the interactive prompt, create the following dictionary that represents an investment of stock: 

>>> t = {
      'name' : 'GOOG',
      'shares' : 100,
      'price'  : 490.10
    }
>>>
Calculate the total cost of this holding: 
>>> cost = t['shares']*t['price']
>>> cost
49010.0
>>>

Change the number of shares to 75. 

>>> t['shares'] = 75
>>> t
