(a) Simple Regular Expression Pattern Matching

In this exercise, we experiment with regular expression pattern matching. First, define the string: 

text = "Guido will be out of the office from 12/14/2012 to 1/3/2013."
Now, define a regular expression pattern to match dates of the form mm/dd/yyyy. 
import re
datepat = re.compile(r'(\d+)/(\d+)/(\d+)')

Now, let's perform some simple regex operations. First, let's search text for all dates and print them out:

>>> for m in datepat.finditer(text):
        print m.group()

12/14/2012
1/3/2013
>>>

Next, let's replace all of the dates with a different date format of the form "yyyy-mm-dd".

>>> def reformat_date(m):
       month,day,year = m.groups()
       return year+"-"+month+"-"+day

>>> datepat.sub(reformat_date, text)
'Guido will be out of the office from 2012-12-14 to 2013-1-3.'
>>>

For simple replacements, you can also specify a replacement string that refers to matching group numbers using "\N". For example, try this: 

>>> datepat.sub(r'\3-\1-\2', text)
'Guido will be out of the office from 2012-12-14 to 2013-1-3.'
>>>

(b) Number Matching (Challenge)

Write a regular expression pattern that matches floating point numbers. For example: 

1.23
0.123
123.
.123
1.23e+2
1.23E+2
1.23e2
1.23e-2
123e+2

Note: This is a lot harder than it looks if you factor in all of the possible placements of the decimal point and forms of exponential notation.
