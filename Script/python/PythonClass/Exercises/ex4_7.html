(a) Install numpy

The numpy extension adds support for arrays and matrices to Python. Your task is simple: install numpy on your machine and try the following example: 

>>> import numpy
>>> a = numpy.array([1,2,3,4])
>>> b = numpy.array([5,6,7,8])
>>> a+b
array([ 6,  8, 10, 12])
>>>

(b) Install matplotlib

The matplotlib extension adds support for plotting. You task: install matplotlib and try making a plot: 

>>> from matplotlib import pyplot
>>> import numpy
>>> x = numpy.linspace(-10,10,1000)
>>> y = 2*numpy.sin(x)+3*numpy.cos(2*x)
>>> pyplot.plot(x,y)
[<matplotlib.lines.Line2D object at 0x12aea370>]
>>> pyplot.show()
>>>
