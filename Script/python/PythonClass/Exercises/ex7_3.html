(a) Using the Debugger

The debugger has the ability to run single Python functions using the pdb.runcall() function. Try using it to single-step through the execution of fieldparse.parse(). 

>>> import fieldparse
>>> import pdb
>>> pdb.runcall(fieldparse.parse, open("Data/portfolio.dat"),[str,int,float],['name','shares','price'])
> PythonClass/fieldparse.py(8)parse()
-> records = []
(Pdb) s
> PythonClass/fieldparse.py(9)parse()
-> for line in lines:
(Pdb) 
... try single stepping by pressing 's' some more...
...
Experiment with some of the other debugger commands such as setting breakpoints, printing code listings, and inspecting values.

(b) Using the profiler

Run one of your earlier programs under the control of the Python profiler. For example: 

% python -m cProfile report.py
Look at the output.

Note: To run the profiler in this manner, you have to launch Python from the command shell (e.g., a Unix terminal or the Windows command shell).
