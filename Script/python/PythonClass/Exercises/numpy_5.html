(a) X-Y plots

Make an X-Y plot of the data from the last exercise. 

>>> import pylab
>>> pylab.plot(xpts,ypts)
[<matplotlib.lines.Line2D object at 0x196c2f50>]
>>> pylab.plot(xpts[:-1],dypts)
[<matplotlib.lines.Line2D object at 0x18ef1e50>]
>>> pylab.show()

Note: In the last step, the number of points in dypts is one less than in xpts so we need to omit one value to make their sizes match.

Try another X-Y plot, but this time with many more points. 

>>> xpts = numpy.linspace(-5,5,1000000)
>>> ypts = xpts*numpy.sin(1.0/xpts)
>>> pylab.plot(xpts,ypts)
[<matplotlib.lines.Line2D object at 0x18eb1e50>]
>>> pylab.show()

Try using the matplotlib viewer to zoom in on the origin.
