Practicum - Exercise 3.3

Modify the parse() function to accept an optional argument missing that allows the user to specify the values to use for fields where the value can't be converted in the input data. For example: 

>>> portfolio = parse(open("Data/portfolio3.dat"),[str,int,float],missing=["",0,0])
>>> for s in portfolio:
       print s
('AA', 15, 39.48)
('AXP', 10, 62.58)
('BA', 5, 98.31)
('C', 0, 53.08)            # Note: Bad shares of '-' replaced by 0
('CAT', 15, 78.29)
...
>>>

In this example, if a ValueError exception occurs while converting any field of data, that field's value should be set to the corresponding value in the missing specifier. If the missing argument is not given, your function should ignore bad input lines as it did before.
