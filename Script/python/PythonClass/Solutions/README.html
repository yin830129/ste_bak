Introduction to Python

The Solutions folder contains complete solution code for most of the class exercises. You should only use this code if you are completely stuck, didn't have enough time to finish one of the exercises, or are simply too worn out to type any further!

To use this solution code, go to the appropriate directory for the exercise in question (for example, 5_2/ for Exercise 5.2). Select ALL of the files in that directory and copy them to the top-level PythonClass folder. This copy process should restore all of the class files to some kind of working state needed for the exercise at hand.

CAUTION: Copying solution code may overwrite any files that you have created yourself. If you want to save your own work, make sure you make a backup first.
