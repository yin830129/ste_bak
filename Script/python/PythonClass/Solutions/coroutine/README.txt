Coroutine Examples:

findco.py:

A simple example of using coroutines to set up a 
processing pipeline.

simplesched.py:

A simple of example of using coroutines to set up
simple tasks.  From the notes.

ipool.py:

An example of using a coroutine to wrap a callback-based
programming interface.  This example might make your
head hurt.

iosched.py:

A more complete example of coroutine task scheduling with
event-based I/O handling and sockets.   This example will
probably make your head explode.    Please refer to 
http://www.dabeaz.com/coroutines for more background
knowledge.
