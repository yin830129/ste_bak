WSGI Examples:

Simple WSGI:
    ctime_wsgi.py     - Simple time service
    kvstore_wsgi.py   - Key-value store

WebOb:
    ctime_webob.py    - Time service
    kvstore_wsgi.py   - Key/value store

Paste:
    dispatch.py       - Dispatch to two different WSGI applications

The file kvstore_wsgi.html is a simple web page for testing
GET and POST operations on the key-value store.
